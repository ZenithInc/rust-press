# アクセスマスク

このページには `access: masked` が設定されています。

オーバーレイはフロントエンドの表示マスクにすぎません。コンテンツを暗号化せず、静的 HTML からコンテンツを削除せず、サーバーサイド認証も提供しません。

## 動作確認

パスワード欄に任意のテキストを入力すると、このブラウザーセッションでオーバーレイが非表示になります。

## 注意

アクセスマスクはフロントエンドのオーバーレイであり、セキュリティ保護ではありません。生成された HTML は直接閲覧できます。
