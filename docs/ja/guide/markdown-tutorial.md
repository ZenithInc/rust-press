# Markdown チュートリアル

このチュートリアルでは、よく使う Markdown 構文と RustPress で有効になっている拡張構文をまとめます。例は `docs/` 配下の `.md` ファイルにそのままコピーできます。

## Frontmatter

各ページの先頭には YAML frontmatter を書けます。タイトル、検索、アクセス方式などのページメタデータを定義します。

```yaml
---
title: ページタイトル
layout: doc
sidebar: true
search: true
access: public
---
```

## 見出し

見出しには `#` を使います。`#` が多いほど深い階層になります。

```markdown
# 見出し 1
## 見出し 2
### 見出し 3
#### 見出し 4
```

RustPress は見出しに安定したアンカーを生成するため、各セクションへ直接リンクできます。

## 段落と改行

空行で区切ると新しい段落になります。

```markdown
これは最初の段落です。

これは次の段落です。
```

同じ段落内で改行したい場合は、行末に半角スペースを 2 つ入れるか `<br>` を使います。

```markdown
1 行目  
2 行目
```

## 強調

`*` または `_` で斜体と太字を表します。`~~` で取り消し線を表します。

```markdown
*斜体*
_斜体_

**太字**
__太字__

***太字斜体***

~~取り消し線~~
```

## リスト

順序なしリストには `-`、`*`、`+` を使います。順序付きリストには数字とピリオドを使います。

```markdown
- 1 つ目
- 2 つ目
  - 子項目
  - 子項目

1. 最初の手順
2. 次の手順
3. 最後の手順
```

## タスクリスト

タスクリストには `- [ ]` と `- [x]` を使います。

```markdown
- [x] 設定を完了する
- [ ] ドキュメントを書く
- [ ] サイトを公開する
```

## リンクと画像

リンクは `[テキスト](URL)`、画像は `![代替テキスト](URL)` で書きます。

```markdown
[ホームへ移動](/ja/)
[CLI ガイド](/ja/guide/cli/)

![Logo](/images/logo.png)
```

リンクや画像にはタイトルも付けられます。

```markdown
[RustPress](/ja/ "ホームへ戻る")
![スクリーンショット](/images/screenshot.png "ページのスクリーンショット")
```

## 引用

引用には `>` を使います。複数行にもでき、入れ子にもできます。

```markdown
> これは引用です。
>
> 引用は複数段落にできます。

> 1 階層目
>> 2 階層目
```

## インラインコード

インラインコードはバッククォートで囲みます。

```markdown
`rust-press build` を実行して静的サイトを生成します。
```

## コードブロック

コードブロックには 3 つのバッククォートを使います。言語名を書くとシンタックスハイライトが有効になります。

````markdown
```bash
rust-press build --config rustpress.toml
```

```rust
fn main() {
    println!("hello");
}
```
````

## 表

表はパイプで列を区切ります。2 行目の `---` がヘッダー区切りです。

```markdown
| 構文 | 用途 |
| --- | --- |
| `#` | 見出し |
| `-` | 順序なしリスト |
| `` `code` `` | インラインコード |
```

コロンで配置を指定できます。

```markdown
| 左寄せ | 中央 | 右寄せ |
| :--- | :---: | ---: |
| A | B | C |
```

## 脚注

脚注は `[^name]` のマーカーと、文書内の定義で書きます。

```markdown
RustPress は脚注をサポートします。[^note]

[^note]: これは脚注の内容です。
```

## 見出し属性

見出しにはカスタム属性を指定できます。よく使うのはカスタム `id` です。

```markdown
## インストール {#install}
```

これにより `/ja/guide/markdown-tutorial/#install` のようなリンクが使えます。

## Mermaid 図

言語が `mermaid` のコードブロックは図としてレンダリングされます。

````markdown
```mermaid
flowchart LR
    A[Markdown を書く] --> B[ビルド]
    B --> C[HTML を生成]
```
````

## 区切り線

3 つ以上の `-`、`*`、`_` で区切り線を作れます。

```markdown
---
***
___
```

## 文字のエスケープ

Markdown の制御文字をそのまま表示したい場合は、前にバックスラッシュを付けます。

```markdown
\# これは見出しではありません
\* これはリストではありません
\[これはリンクではありません\](https://example.com)
```

## HTML

Markdown には少量の HTML を直接書けます。Markdown で表現できない場合だけ使うことをおすすめします。

```html
<kbd>Shift</kbd>
<br>
<span class="custom">カスタム内容</span>
```
