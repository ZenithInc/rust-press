# 테마

기본 테마는 정적 HTML, CSS, 작은 JavaScript runtime으로 구성됩니다.

## 색상 모드

테마는 두 가지 기본 색상 모드를 제공합니다.

- `light`
- `dark`

`allow_switch = true`이면 상단 바에 Light/Dark 전환기가 표시되고 선택한 모드가 `localStorage`에 저장됩니다.

## 레이아웃

생성된 페이지에는 다음이 포함됩니다.

- sticky 상단 내비게이션
- 사이드바 내비게이션
- 반응형 모바일 메뉴
- 레벨 2 및 레벨 3 제목용 목차
- 로컬 검색 대화상자

테마는 프런트엔드 마스크를 보안으로 설명하지 않습니다.
